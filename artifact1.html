Artifact Description

This artifact is the Grazioso Salvare Dashboard, a client/server web application developed in CS-340. The application analyzes and visualizes data from the Austin Animal Center Outcomes dataset to support animal rescue operations and assist stakeholders in making informed decisions. Users can filter and analyze data based on breed, age, and outcome type. The system uses Python, Dash, and Plotly for the frontend, and MongoDB as the backend database.

Enhancements Implemented

The original application provided basic functionality but required improvements in maintainability, scalability, and reliability. To address this, I refactored large functions into modular components and created a dedicated AnimalShelter class to manage database operations.

I implemented structured logging, improved error handling, and added input validation to ensure data integrity. Additionally, I enhanced database queries and introduced filtered query functionality to support dynamic user interaction and improve performance.

Enhanced Code (AnimalShelter.py)


from pymongo import MongoClient
import logging

logging.basicConfig(level=logging.INFO)

class AnimalShelter:

    def __init__(self, username="aacuser", password="ChangeMe123"):
        try:
            self.client = MongoClient(
                f"mongodb://{username}:{password}@localhost:27017/?authSource=admin"
            )
            self.database = self.client["aac"]
            self.collection = self.database["animals"]

            logging.info("MongoDB connected successfully!")

        except Exception as e:
            logging.error(f"Connection failed: {e}")

    def validate_data(self, data):
        if not data or not isinstance(data, dict):
            raise ValueError("Invalid data")

        if "animal_type" not in data or "breed" not in data:
            raise ValueError("Missing required fields")

        return True

    def create(self, data):
        try:
            self.validate_data(data)
            result = self.collection.insert_one(data)
            logging.info("Insert successful")
            return result.inserted_id
        except Exception as e:
            logging.error(f"Insert failed: {e}")
            return None

    def read_filtered(self, animal_type=None, breed=None):
        query = {}

        if animal_type:
            query["animal_type"] = animal_type
        if breed:
            query["breed"] = breed

        logging.info(f"Running query: {query}")
        return list(self.collection.find(query))
